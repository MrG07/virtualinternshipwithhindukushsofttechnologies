

Asalamualaiikum, on the Day 32 of this Amazing Virtual Internship


Is task mein hum ne React ke basic concepts seekhe, khas taur par components aur props. React mein 
har web page ko chhoti chhoti reusable parts mein tod diya jata hai, jinhein components kehte hain.

Hum ne teen components banaye: Header.js, Footer.js, aur UserCard.js. Ye sab hum ne main file App.js 
mein import kiye aur HTML tags ki tarah use kiya jaise <Header />, <Footer />, aur <UserCard />.

Props ka use data ko ek component se doosre component tak bhejne ke liye hota hai. Jaise:

<UserCard name="Saood Masood" age={17} address="Ayun, Chitral" />

Yahan name, age, aur address ko props ke zariye UserCard component mein bheja gaya, jahan unhein {props.name}
aur {props.age} ke through display kiya gaya.

Is task se humein ye samajh aaya ke React ka component system code ko reusable aur manage karne mein asaan banata hai. 
Har part apna kaam alag karta hai, jisse project clean aur professional lagta hai.
